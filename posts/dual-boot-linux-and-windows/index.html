Dual Boot Linux and Windows

January 05, 2022

Steps:

  1. Figure out UEFI or legacy BIOS
    System Information > System Summary > BIOS Mode: UEFI

  2. Disable Fast Startup
    Control Panel > Hardware and Sound > Power Options > System Settings > Choose what the power buttons do and uncheck the Turn on fast startup box.

  3. Disable Secure Boot
    Disable Secure Boot for Aorus is different, In BIOS, Enable CSM which means Secure Boot is disabled.

  4. Shrink volume to save free space for Linux

  5. Create USB disk by UUI

  6. Install Ubuntu

  7. Installation type: Something else

  8. Partition:

    Partition Purpose Size
    / root partiotion for linux 20 G
    swap area swap area double of RAM 64g
    /home home partition the rest of space

  9. Wait for installation finish

  10. Press F12 to change Boot order to start with Linux
