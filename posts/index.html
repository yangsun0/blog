Dependency Injection

January 31, 2020
Dependency injection is supper useful. The usage is so transparent, I’m wondering how it works? And what about the performance? How to use dependency injection Why dependency injection is useful? The most important benefit is decouple. For example, A dependents on B, B dependents on C. It’s like a path A -> B -> C. One day, C add a new dependency D. Everyone on the path need to change. Dependency injection handles this problem very well.
